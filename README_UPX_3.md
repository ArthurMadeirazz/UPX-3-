# UPX 3 - Sistema de Gestao do Custo e Desempenho de Placas Solares

Projeto academico desenvolvido na disciplina de Usina de Projetos Experimentais 3 (UPX 3), com o objetivo de simular os custos, beneficios e impacto ambiental do uso de energia solar.

O projeto foi apresentado em seminario academico com banner e prototipo fisico. A apresentacao recebeu avaliacao maxima, reforcando a qualidade da proposta e da demonstracao realizada.

## Sobre o projeto

A proposta surgiu a partir de um problema real: muitas pessoas tem interesse em energia solar, mas nao sabem estimar de forma simples quantas placas seriam necessarias, quanto poderiam economizar e em quanto tempo o investimento poderia retornar.

Para resolver isso, foi desenvolvido um simulador web que recebe dados de consumo mensal e valor medio da conta de luz. A partir dessas informacoes, o sistema calcula estimativas de economia, quantidade de placas solares necessarias, retorno do investimento e reducao de emissao de CO2.

O trabalho tambem contou com um prototipo fisico usando Arduino para representar, de forma pratica, conceitos relacionados a geracao e monitoramento de energia.

## Objetivo

Criar uma solucao interativa que ajude o usuario a entender, de forma simples, os beneficios da energia solar, combinando:

- Simulador web para calculos energeticos e financeiros
- Interface visual simples e responsiva
- Prototipo fisico com Arduino para demonstracao pratica
- Indicadores de economia, retorno e impacto ambiental

## Funcionalidades do simulador

- Entrada do consumo mensal em kWh
- Entrada do valor medio da conta de luz
- Calculo da quantidade estimada de placas solares
- Estimativa de economia mensal
- Estimativa de economia anual
- Calculo aproximado do tempo de retorno do investimento
- Estimativa de reducao de CO2
- Comparacao ambiental com quantidade aproximada de arvores plantadas
- Atualizacao dos resultados em tempo real conforme os dados sao alterados

## Prototipo fisico

Durante o desenvolvimento academico, o projeto tambem utilizou componentes de hardware para demonstracao pratica, incluindo:

- Arduino Uno
- Protoboard
- LEDs
- Resistores
- Potenciometro
- Sensores e modulos de apoio
- Cabos jumpers

O Arduino foi utilizado como parte da demonstracao do funcionamento do sistema, representando alertas e respostas visuais relacionadas aos dados simulados.
